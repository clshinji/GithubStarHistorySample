# ユーザがテストして気付いたこと

- GitHubスター数が40,000件までは連続的な値として取得できているが、40,000件を超えると、その次のデータは今日時点のスター数になっていて連続的なデータが取得できていない
- スターの取得に時間がかかり過ぎている。（待ち時間が長い）
- マウスオーバーしたときの吹き出しが、マウスの場所とズレている
- 棒グラフ、積み上げ棒グラフが表示されない
- Log scale にしたとき、Y軸のデフォルトの最小値が `1` になっているが、スター数の最小値に併せてオートスケールした方が良い。例えば、スター数の最小値が `1100` ならY軸のデフォルトの最小値は `1000` にしたり、スター数の最小値が `800` ならY軸のデフォルトの最小値は `100` にする。
- 画面のカラーを、ダークモードとライトモードに対応してほしい。ブラウザのデフォルトに合わせてくれると更に良い。
