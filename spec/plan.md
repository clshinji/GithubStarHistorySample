# Github star history メーカー

## 概要

ユーザは好きなGitHubリポジトリを複数選択して、スター数を時系列でグラフ化できるブラウザツール

## 参考にしたサイト

[Star History](https://www.star-history.com/)

## 機能

- ユーザがGitHubリポジトリのURLを複数入力する
- ユーザが集計期間の開始年月日と終了年月日を指定する
- 指定されたGitHubリポジトリのスター数の履歴を取得する。
  - 取得する期間は、ユーザが指定した期間
  - リポジトリが作成される前のデータはグラフには表示しなくて良い
- グラフでは、リポジトリごとに区別して可視化出来るようにする
- 作成したグラフはユーザが加工できるようにする
  - グラフの種類
    - デフォルトでは折れ線グラフ（プロットなし）
    - 変更可能なグラフの種類
      - 折れ線グラフ（デフォルト）
      - 棒グラフ（積み上げ）
      - 棒グラフ（積み上げなし）
  - 編集できる要素
    - 横軸（時間）の表示範囲（最大値を変えたり、最小を変えたり）
    - 縦軸（スター数）の表示範囲（最大値を変えたり、最小を変えたり）
    - グラフの要素の色を変えられる
      - 折れ線グラフであれば、線の色をリポジトリごとに好きな色に変更できる
      - 棒グラフであれば、棒の色をリポジトリごとに好きな色に変更できる

## 技術要件

- HTML+JavaScript+CSS で作る
- GitHub Pages で公開できるようにする
