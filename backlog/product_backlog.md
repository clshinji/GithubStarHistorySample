# プロダクトバックログ - GitHub Star History メーカー

GitHub Star History メーカーの価値を向上させるためのアイテムを管理します。優先順位（Prio）は A（最高）〜 C（低）で示します。

| Prio | カテゴリ | アイテム名 | 概要 | 状態 |
| :--- | :--- | :--- | :--- | :--- |
| **A** | 可視化 | 画像エクスポート | グラフをPNG/SVG形式でダウンロードできる機能 | 進行中 |
| **A** | 可視化 | CSVエクスポート | 取得した時系列データをCSVで保存できる機能 | 進行中 |
| **A** | 分析 | 対数軸 (Log scale) | スター数の差が激しいリポジトリ比較のための対数軸表示 | 進行中 |
| **A** | 分析 | 開始点同期 (Align timeline) | 「作成日」を基準(Day 0)として成長速度を比較する機能 | 進行中 |
| **B** | データ | 4万件制限の完全回避 | GitHub APIの制限を超えたデータ取得手法の導入 | 未着手 |
| **B** | 利便性 | URLパラメータ共有 | 追加済みリポジトリをURLに含め、シェア可能にする機能 | 未着手 |
| **B** | 利便性 | データの永続キャッシュ | IndexedDB 等を活用したブラウザ内キャッシュ | 未着手 |
| **B** | UI/UX | ツールチップの強化 | ホバー時の詳細データ表示と凡例による表示切り替え | 未着手 |
| **C** | UI/UX | xkcdテーマ | 手書き風デザインのサポート | 未着手 |
| **C** | UI/UX | ダークモード | システム設定に応じたUIのダーク化 | 未着手 |
| **C** | 分析 | 多角的な指標 | スター以外の指標（Fork, Issue, 増加率等）のサポート | 未着手 |
