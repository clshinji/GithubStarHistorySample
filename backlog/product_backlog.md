# プロダクトバックログ - GitHub Star History メーカー

GitHub Star History メーカーの価値を向上させるためのアイテムを管理します。優先順位（Prio）は A（最高）〜 C（低）で示します。

| Prio | カテゴリ | アイテム名 | 概要 | 状態 |
| :--- | :--- | :--- | :--- | :--- |
| **A** | データ | 4万件制限の回避 | 4万件以降も滑らかなグラフを描画するための二分探索サンプリングの導入 | 進行中 |
| **A** | データ | データ取得の高速化 | 並列Fetchやサンプリング最適化による待ち時間の短縮 | 進行中 |
| **A** | 不具合 | 描画バグ修正 | 棒グラフの非表示、ツールチップのズレの解消 | 進行中 |
| **A** | 分析 | Log scale 改善 | スター数に応じたY軸最小値のオートスケール実装 | 進行中 |
| **B** | UI/UX | ダークモード対応 | ブラウザ設定に連動したライト/ダークテーマの切り替え | 進行中 |
| **B** | 利便性 | URLパラメータ共有 | 追加済みリポジトリをURLに含め、シェア可能にする機能 | 未着手 |
| **B** | 利便性 | データの永続キャッシュ | IndexedDB 等を活用したブラウザ内キャッシュ | 未着手 |
| **C** | UI/UX | xkcdテーマ | 手書き風デザインのサポート | 未着手 |
| **C** | 分析 | 多角的な指標 | スター以外の指標（Fork, Issue, 増加率等）のサポート | 未着手 |
